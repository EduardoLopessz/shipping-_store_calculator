# shipping-_store_calculator
Calculadora de frete em Python com sistema de login, validação de dados e menu interativo. O programa calcula o valor do frete com base no peso do produto e no estado de destino, além de oferecer opções de ajuda e simulação de rastreamento. Utiliza estruturas como while, if/elif e entrada de dados para criar um fluxo funcional e contínuo.
